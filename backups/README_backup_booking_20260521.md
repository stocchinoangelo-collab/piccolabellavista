# Piccola Bellavista

Sito statico leggero per la casa vacanza Piccola Bellavista, a Pirri, Cagliari.

## Come avviare

Apri direttamente `index.html` nel browser.

In alternativa, dalla cartella del progetto:

```powershell
python -m http.server 8080
```

Poi apri:

```text
http://127.0.0.1:8080
```

## Struttura

- `index.html` - pagina principale del sito.
- `styles.css` - grafica e layout.
- `script.js` - selettore lingua Italiano/Inglese.
- `assets/` - loghi e immagini copiate dai materiali originali.
- `originali_backup/` - copie dei materiali trovati nei Download.

## Traduzione

Il sito include un selettore lingua `IT/EN` nella testata.
La lingua scelta viene ricordata dal browser per le visite successive.
